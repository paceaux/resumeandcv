I purchased frankmtaylor.com about eight years ago when I began a job search. This site was always my playground, first. Building and rebuilding this site helped me learn what front-end development was all about. The added bonus was that I could also make it my resume..

Code

With the exception of the normalize, I wrote all of the code you see here, by hand, using Visual Studio Code. Previous versions were built with Sublime Text or notepad.

This site was built using Gulp, Assemble.io (which includes some Handlebars), and spare time.

Icons

Most icons come from Simple Icons. The JSFiddle logo comes from SVG Repo.
